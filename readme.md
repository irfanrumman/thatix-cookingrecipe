# Thatix Cooking Recipe

A modern and responsive recipe website built with **HTML**, **CSS**, and **JavaScript**, designed to provide an engaging and user-friendly cooking experience.

## 🌐 Live Demo

- **Live Site:** https://irfanrumman.github.io/thatix-cookingrecipe/

## 📂 Repository

- **GitHub:** https://github.com/irfanrumman/thatix-cookingrecipe

---

## ✨ Features

- Modern and responsive user interface
- Clean and intuitive navigation
- Fully responsive layout for desktop, tablet, and mobile devices
- Well-structured and maintainable codebase
- Attractive recipe cards and content sections
- Smooth browsing experience across different screen sizes

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/irfanrumman/thatix-cookingrecipe.git
```

Navigate to the project directory:

```bash
cd thatix-cookingrecipe
```

Open the project:

Simply open the `index.html` file in your browser, or use the **Live Server** extension in VS Code.
